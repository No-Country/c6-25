Frontend

Front End es la parte de una aplicación que interactúa con los usuarios, es conocida como el lado del cliente. Básicamente es todo lo que vemos en la pantalla cuando accedemos a un sitio web o aplicación: tipos de letra, colores, adaptación para distintas pantallas,los efectos del ratón, teclado, movimientos, desplazamientos, efectos visuales y otros elementos que permiten navegar dentro de una página web. Este conjunto crea la experiencia del usuario.

Backend

Backend nos referimos al interior de las aplicaciones que viven en el servidor y al que a menudo se le denomina “el lado del servidor”. El back end del sitio web consiste en un servidor, una aplicación y una base de datos. Se toman los datos, se procesa la información y se envía al usuario. Los desarrolladores de Front end y Back end suelen trabajar juntos para que todo funcione correctamente.

Diseño Ux Ui

Un UI/UX Designer o un Diseñador UI/UX es el profesional que se encarga de supervisar la experiencia de usuario, la UX (User Experience), es decir, el modo en el que los usuarios interactúan con una web o una app, o incluso con un producto o servicio. La clave de su trabajo está en conocer a esos usuarios: sabiendo cuáles son sus gustos o lo que necesitan es mucho más sencillo elaborar la estrategia necesaria para dar respuesta a esas necesidades de la manera más satisfactoria.
